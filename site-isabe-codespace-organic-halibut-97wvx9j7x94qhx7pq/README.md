# Educação que Cultiva o Futuro

Site desenvolvido para o projeto escolar com o tema:

**Papel da educação no futuro sustentável do agro**

## Sobre o projeto

Este site tem como objetivo mostrar como a educação pode contribuir para um futuro mais sustentável no agro, aproximando conhecimento, tecnologia, consciência ambiental e participação social.

A proposta é apresentar o tema de forma didática, acessível e interativa, mostrando que a sustentabilidade no campo não depende apenas de máquinas ou produção, mas também da formação de pessoas conscientes e preparadas.

## Objetivo principal

O principal objetivo do site é explicar o papel da educação na construção de um agro mais sustentável, destacando ações que podem ser aplicadas em diferentes ambientes, como:

- escola;
- casa;
- comunidade;
- propriedade rural.

## Funcionalidades do site

O site possui:

- Página inicial com apresentação do tema;
- Seções explicativas sobre educação, sustentabilidade e agro;
- Mapa de Ações Sustentáveis;
- Quiz educativo;
- Área de projeto escolar;
- Layout responsivo para computador e celular;
- Rodapé com copyright e dados do aluno.

## Função especial: Mapa de Ações Sustentáveis

A principal função interativa do site é o **Mapa de Ações Sustentáveis**.

Nele, o usuário escolhe um ambiente e recebe sugestões de atitudes sustentáveis relacionadas ao agro e à educação.

Essa função ajuda a transformar o conteúdo do site em algo mais prático, mostrando como pequenas ações podem contribuir para uma produção rural mais consciente e responsável.

## Tecnologias utilizadas

O projeto foi desenvolvido com:

- HTML5;
- CSS3;
- JavaScript.

Não foram utilizados frameworks externos, tornando o site simples, leve e fácil de executar.

## Como executar o projeto

Para abrir o site localmente:

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto.
3. Clique duas vezes no arquivo `index.html`.

O site abrirá diretamente no navegador.
